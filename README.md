# Rock Paper Scissors AI Tournament

This project is an advanced console-based Rock Paper Scissors game implemented in Python.  
Unlike the basic version, this game includes a best-of-five tournament format, an AI opponent with adaptive strategy, score tracking, and player move statistics.

The project is designed to demonstrate logical decision-making, basic AI behavior, and clean program structure.

---

## Game Rules

- The game is played in a best-of-five format.
- The player chooses between rock, paper, or scissors.
- The AI analyzes the player's previous moves and tries to counter the most frequent choice.
- A small amount of randomness is added to avoid predictability.

---

## Features

- Best of five tournament system
- AI strategy based on player move frequency
- Score tracking for each round
- Player statistics for move usage
- Clear and readable console output
- Beginner-friendly and interview-safe logic

---

## Technologies Used

- Python 3
- Random module
- Dictionaries for statistics tracking
- Conditional logic

---
